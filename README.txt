Script de automatización de auditorias de seguridad en sistemas Linux. 

Este script automatiza la recolección de infromacion relevante para auditorias de seguridad en sistemas Linux 

Objetivos:
- Automatizar la recolección de información de seguridad en sistemas Linux.
- Generar informes que ayuden en auditorias y análisis de seguridad
- Mejorar la eficiencia de tareas repetitivas 

Caracteristicas:

- Recopila información del sistema
- Lista usuarios y grupos, incluyendo aquellos con privilegios sudo
- Enumera procesos y servicios en ejecución. 
- Muestra conexiones de red y puertos abiertos. 
- Verifica permisos de archivos críticos. 
- Obtiene la configuración del firewall. 
- Verifica actualizaciones pendientes. 

Requisitos: 

- Sistema operativo Linux. 
- Privilegios de administrador (root). 

Advertencia:

Este script debe ser utilizado únicamente en sistemas donde tengas autorización para realizar auditorias de seguridad. 

Uso: 

1. Clona este repositorio o descarga el script 'auditoria_seguridad.sh'.
2. Otorga permisos de ejecución al script: #chmod +x auditoria_seguridad.sh
